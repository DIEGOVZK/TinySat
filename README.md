# TinySat
TinySat is an ESP8266 library that allows you to build a small satellite for educational purposes.

## üõ∞ TinySat
The "TinySat" project is an introductory approach to satellite systems for educational purposes. The satellite telemetry system captures the data from the HC-SR04 sensor, which is sent through a local network to the database via the [MKSENSE](https://github.com/DIEGOVZK/MKSENSE) app, which manages the data and connection requests, and stores it in a database. Then, the data can be used to generate graphs to exemplify mission results. The data shows the distance between the smallsat and the ground, simulating missions like the "AHN DEM Netherlands" earth elevation scanning missions.

## üèó Casing
The [case](https://github.com/DIEGOVZK/TinySat/blob/main/TinyCase) is an MDF laser cut casing that houses all the electronics for the project. This can be made in a 60x50cm sheet of MDF wood and will be able to make 9 (nine) replicas of the TinySat cube.
